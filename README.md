🚀 Galaxy Shooter

A fast-paced 2D space shooter game built with Python and Pygame. Control your spaceship, dodge enemy attacks, and destroy incoming enemy ships to achieve the highest score possible.

---

✨ Features

- 🚀 Smooth spaceship movement
- 👾 Continuously spawning enemy ships
- 🔫 Shooting mechanics
- 💥 Collision detection
- 🏆 Score tracking
- 🔊 Sound effects
- ▶️ Start screen with "Press Enter to Play"
- ⏸️ Pause and resume functionality
- ☠️ Game Over screen
- 🎮 Responsive and enjoyable gameplay

---

🛠️ Technologies Used

- Python 3
- Pygame

---

🎮 Controls

Key| Action
Enter| Start the Game
← / → Arrow Keys| Move Left / Right
Space| Shoot
P| Pause / Resume the Game
Esc| Quit the Game

---

🚀 Getting Started

Clone the repository

git clone https://github.com/1nsaned/Galaxy-Shooter.git

Install Pygame

pip install pygame

Run the game

python main.py
