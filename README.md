Galaxy Shooter 🚀

A 2D space shooter game built with Python and the Arcade library.

Features

- 🚀 Player spaceship with WASD movement
- 🎯 Mouse aiming and shooting
- 👾 Multiple enemy spaceships
- ❤️ Health system
- 💚 Health pickup power-up
- 📊 Score system
- 📈 Enemy health bars
- ⏸️ Pause menu
- ▶️ Start menu
- 🔄 Restart after Game Over
- 🎵 Background music and sound effects

Controls

- WASD - Move
- Mouse - Aim
- Left Click / Space - Shoot
- P - Pause/Resume
- R - Restart after Game Over
- ENTER - Start Game

Technologies Used

- Python
- Arcade

How to Run

1. Install Python.
2. Install Arcade:
   pip install arcade
3. Run:
   python spaceshootinggame.py
